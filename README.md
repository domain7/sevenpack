# Features:
- ES6 module bundling and minification
- ESLint
- Custom modernizr build
- SCSS compilation and minification
- Autoprefixer
- SVG symbol sprite generation
- Livereload

# Setup
`npm install`

# Run
`npm start`

# One time commands

## Generate SVG sprite
`npm run svg`

## Generate custom modernizr
`npm run modernizr`
